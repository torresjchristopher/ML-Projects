# ML-Projects

A consolidated repository of machine learning projects covering various technologies and use cases.

## Projects

1. **nlp-fine-tuning-api** - Fine-tune Transformer models for text classification with API deployment
2. **ml-pipeline** - End-to-end ML pipeline with MLOps for tabular data
3. **fastapi-ml-deployment** - ML model deployment with FastAPI
4. **ruby-ml-sinatra** - Ruby Sinatra web app with ML backend
5. **rust-data-preprocessing** - Real-time data preprocessing engine in Rust
6. **retail-sales-forecaster** - Time series forecasting for retail sales
7. **resume-matcher** - Resume matching using transformers and similarity search
8. **sql-python-rdms** - Relational database management system

Each subdirectory contains a complete project with its own documentation, environment setup, and deployment configurations.
